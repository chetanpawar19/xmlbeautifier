The XMLBeautifier

The XMLBeautifier helps to beautify your unformatted ugly xml code. the application is powered by Django and python.

Just put your code into text box and click on beautify button, XMLBeautifier will produce formatted code. however, the applicaton

contains server side interactions, so please do not submit sensitive information. note- the XMLBeautifier backend code

is developed by the developer and do not uses any third party libraries.
